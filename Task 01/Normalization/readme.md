## Redundancy
> একই ধরনের ডাটা বা Duplicate Value যদি একটা টেবিলে Multiple থাকে সেইটা হলো Redundancy.
  
## Normalization
> একটা টেবিলে এ যদি Redundancy থাকে সেইটাকে Reduce বা Remove করার উপায় হলো Normalization।

<br>

![Image](https://res.cloudinary.com/dxrdnpp47/image/upload/v1766244838/Screenshot_31_nxbwzg.png)

```
এইখানে Row এর মধ্যে যে Redundancy ছিল তা Employ ID কে Primary Key করে দিলে Redundancy Remove হয়ে যায়।
```

<br>

![Image](https://res.cloudinary.com/dxrdnpp47/image/upload/v1766245079/Screenshot_32_tf8vkb.png)

```
এইখানে Column এর মধ্যে যে Redundancy ছিল তা Remove করা যায় না।
Reduce করা যায়।
```
