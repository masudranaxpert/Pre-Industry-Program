## Python-এ যেকোনো অবজেক্ট যদি নিচের ম্যাজিক মেথডগুলো ইমপ্লিমেন্ট করে, তাহলে সে ডিকশনারির মতো ব্যবহার করা যায়:
- `__getitem__()` → `obj[key]`
- `__len__()` → `len(obj)`
- `keys()`, `items()`, `values()` → ডিক্টের মতো মেথড
`ErrorDict` এই সব ম্যাজিক মেথড ইমপ্লিমেন্ট করেছে।
